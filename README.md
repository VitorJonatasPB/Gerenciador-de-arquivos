# Gerenciador de Arquivos em Java

Um projeto simples em Java para gerenciar arquivos via console, com menu interativo e registro de logs.

## Funcionalidades
- Criar arquivos
- Listar arquivos
- Ler arquivos
- Renomear arquivos
- Deletar arquivos
- Buscar palavra-chave em arquivos
- Log automático das ações (log.txt)
